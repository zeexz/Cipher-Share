# Contributing to CipherShare

Thank you for your interest in contributing to CipherShare! We welcome contributions to make this project better and more secure.

## How to Contribute

1.  **Fork the repository**: Click the "Fork" button on the top right of this page.
2.  **Clone your fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/cipher-share-api.git
    ```
3.  **Create a branch**:
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Make your changes**: Implement your feature or fix.
5.  **Commit your changes**:
    ```bash
    git commit -m "Add some amazing feature"
    ```
6.  **Push to your fork**:
    ```bash
    git push origin feature/amazing-feature
    ```
7.  **Open a Pull Request**: Go to the original repository and click "New Pull Request".

## Reporting Bugs

If you find a security vulnerability or a bug, please open an issue or contact the maintainers directly if it is sensitive.

## Style Guide

- Follow standard Java coding conventions.
- Ensure your code is formatted cleanly.
- Add comments for complex logic.
